# Wallpaper Downloader
___

This is a small application that is used to download the daily "image of the day" from NASA. 
You can use my api. But if you are going to be using this with the automatic download 
I am going to ask that you make your own account on the NASA website and use your api instead

### Installation
___

You need to decide where you want this code to be downloaded to
make a new folder and name it whatever you like
after that make sure you have a version of python on your computer 
https://www.python.org/downloads/

make sure you have git installed on your computer as well
https://git-scm.com/install/

Now navigate to you the folder you want to use with
cd <your filepath>

Get the project
git clone https://github.com/SilasNight/wallpaper-downloader

Create a virtual enviroment for the project
python3 -m venv .venv

Navigate to the project
cd wallpaper-downloader

Install the requirements
pip install -r requirements.txt

Run the program
py main.py
